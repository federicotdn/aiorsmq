AIORSMQ Documentation
=====================

This is the documentation for aiorsmq (**A**\ sync **IO** **RSMQ**), an
asynchronous (async/await) implementation of `RSMQ
<https://github.com/smrchy/rsmq>`_ for Python 3.6+. It aims to provide
all the features that RSMQ provides, but for Python users.
